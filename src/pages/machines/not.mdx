Suppose you have a machine *M* that accepts every string with some
property *P*. You want a machine that accepts every string *without*
that property — every string *M* rejects.

The trick is almost embarrassing. Take *M*, change nothing about
the wiring, and **swap which states are accepting.** Every double
circle becomes a single circle; every single circle becomes a
double circle.

Below, on the left, is the machine that accepts when the count of
`1`s is divisible by 3. On the right is the very same wiring with
the accepting set flipped — it accepts when the count is *not*
divisible by 3. Type the same input into both and watch the
verdicts disagree.

Notice that the two machines are *literally* the same drawing, save
for which circles are double-bordered. They will always disagree on
every input, because between them they cover every possible final
state.

This costs no new wiring and no new states. You get **NOT** for
free, on any machine you have already built.
