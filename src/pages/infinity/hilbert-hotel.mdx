This idea of countability is famously illustrated by **Hilbert's
Hotel**. It's a hotel with an infinite number of rooms, numbered
$1, 2, 3, \ldots$. On a particular night, the hotel is completely
full — every room has a guest.

- *A new guest arrives.* The manager isn't worried. He asks every
  guest in room $n$ to please move to room $n + 1$. Room $1$ is
  now free for the new arrival. Everyone still has a room.

- *An infinite bus arrives.* A bus with a countably infinite
  number of new passengers arrives. Now what? The manager has a
  clever solution. He asks every guest in room $n$ to move to
  room $2n$. This clears out all the odd-numbered rooms
  ($1, 3, 5, \ldots$). He then places the passenger from seat $k$
  on the bus into room $2k - 1$. Again, everyone is accommodated.

- *Infinitely many infinite buses arrive.* This is the ultimate
  challenge. We have buses $B_1, B_2, B_3, \ldots$, each with
  passengers $P_1, P_2, P_3, \ldots$. To assign a room to passenger
  $P_j$ from bus $B_i$, we need to find a unique room number for
  the pair $(i, j)$. But we just solved this with our second frog
  puzzle! We can use the diagonal enumeration of
  $\mathbb{N} \times \mathbb{N}$ to create a single list of all
  passengers from all buses and assign them rooms $1, 2, 3, \ldots$
  in that order.

The lesson of Hilbert's Hotel is that if a set is countably
infinite, it can always be "shifted around" to make room for
another countably infinite set.

}}>Exercise 1 — A fleet of infinite-decker buses</div>

Welcome to the Grand Hilbert Hotel, with rooms numbered
$1, 2, 3, \ldots$. On this fateful evening, the hotel is
completely full. Then an entire fleet of buses appears, also
numbered $1, 2, 3, \ldots$. These are no ordinary buses. Each one
is an "Infinite Decker Hilbert Bus." Bus $i$ has decks numbered
$1, 2, 3, \ldots$, and on each deck of each bus there are seats
numbered $1, 2, 3, \ldots$. To the manager's horror, every single
seat, on every single deck, on every single bus, is occupied by a
new patron.

Can the brilliant night manager devise a plan to move the existing
guests and accommodate every single new patron from this infinite
fleet of infinite buses? If so, explain the manager's
re-assignment plan.
</div>

show that a humbler goal can be achieved. Let $\mathcal{F}$ be the
collection of all *finite* reading lists. Show that we can write

which feature exactly $n$ pages. Show that $\mathcal{F}_n$ is
